CSS Reference Guide

รวมตัวอย่าง 10 คำสั่งพื้นฐาน พร้อมการแสดงผล

1. Color

ใช้เปลี่ยนสีตัวอักษร: color

ข้อความนี้ใช้สี Hex #e74c3c

2. Background Color

เปลี่ยนสีพื้นหลัง: background-color

พื้นหลังสีน้ำเงิน

3. Font Size

ปรับขนาดตัวอักษร: font-size

ขนาด 24px

4. Border

สร้างเส้นขอบ: border

กล่องมีเส้นขอบสีเขียวหนา 3px

5. Border Radius

ทำมุมโค้งมน: border-radius

มุมโค้งมน 20px

6. Padding & Margin

ระยะห่าง: padding (ใน) margin (นอก)

กล่องที่มี Padding ภายใน

7. Width & Height

กำหนดขนาด: width & height

100x100

8. Text Align

จัดวางข้อความ: text-align

ชิดขวา (Right)

9. Box Shadow

เพิ่มเงาให้วัตถุ: box-shadow

กล่องมีเงาฟุ้ง

10. Opacity

ค่าความโปร่งใส: opacity

โปร่งใสระดับ 0.4
